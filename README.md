# Sprint Story Estimation (Planning Poker)

Streamlit-based planning poker app for sprint story estimation.

## Features

- Fibonacci story points: 1, 2, 3, 5, 8, 13, 21, plus ☕ ("cup")
- Users enter their name to join
- Estimates are hidden until the Scrum Master reveals
- Participant list updates automatically (shows who is still waiting)

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy (Streamlit Community Cloud)

1. Push this repo to GitHub
2. Go to https://share.streamlit.io
3. Deploy with main file path: `app.py`
